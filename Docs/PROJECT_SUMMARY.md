# Project Summary

AI Trip Agent demonstrates a practical AI application workflow rather than a static prompt demo. It combines a conversational UI, streamed backend planning, tool-assisted evidence collection, structured itinerary generation, and map-based visualization.

## What It Shows

- Full-stack TypeScript and Python engineering.
- Agent orchestration with LangGraph.
- Retrieval-augmented generation with pgvector.
- Robust streaming UX through SSE.
- Session persistence and rate limiting with Redis.
- Dockerized local infrastructure.
- Automated checks for backend and frontend code.

## Resume Highlights

- Built a full-stack AI travel planning assistant with Next.js, FastAPI, LangGraph, PostgreSQL/pgvector, and Redis.
- Implemented SSE streaming for real-time agent reasoning, tool calls, and structured itinerary output.
- Designed itinerary quality checks, budget summary, export flow, and AMap-based route visualization.
- Added CI, linting, type checking, backend tests, and frontend E2E coverage for a public-ready engineering repository.
